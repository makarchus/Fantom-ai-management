Getting started

  1. Create an account — email and password on the home page.
  2. Verify your email — enter the 6-digit code from your inbox. Match the 2-letter prefix shown on screen (e.g. K7) to the email subject [K7].
  3. Save your recovery key — shown once. You only need it if you lose email access.
  4. Connect Fathom — open Settings and add your Fathom API key.
  5. Refresh meetings — click Refresh in the header to pull recordings from Fathom.

Sign in & security

Every sign-in uses two-factor authentication:

  • Enter email and password (or Sign in with Google).
  • Check your email for a code with a unique prefix.
  • Enter the 6 digits on the verification screen.
If you received multiple codes, use only the one whose prefix matches what is displayed on the sign-in screen.
Lost email access? Use Recover with encryption key on the sign-in page with the recovery key you saved at signup.

Connect Fathom

Fathom records your meetings. Meeting Intelligence imports summaries and transcripts to extract action items.

  1. In Fathom: Settings → API Access — generate an API key.
  2. In Meeting Intelligence: Settings (gear icon) — paste your API key and recorder email.
  3. Click Refresh in the header to sync meetings.

Meetings appear under the Fathom tab in the sidebar. Import a meeting to save it permanently and extract action items.

Meetings & folders

Sidebar tabs

  • Fathom — recordings from your Fathom account (sync on Refresh).
  • Saved — imported meetings stored in your encrypted vault.

Folders

Organize meetings into folders (Sales, Engineering, 1:1s, etc.). Drag or use folder controls to categorize. Use Organize for AI-assisted sorting.

Meeting detail

Open a saved meeting to view:

  • Summary — AI-generated meeting recap
  • Transcript — full conversation
  • Actions — extracted action items grouped by assignee

Action items

Action items are extracted when you import a meeting. Each item has:

  • Description, priority (high / medium / low), due date
  • One or more assignee emails
  • Progress comments (running history)

As meeting owner

  • Edit — change description, priority, due date, assignees, notes.
  • Progress comments — add updates visible to all assignees.
  • Mark complete (✓) — moves item to Archive.
  • Delete (trash) — permanently removes the item and notifies removed assignees.

As assignee

When someone assigns you by email, items appear in My Actions and the Action Queue (right panel). Add progress comments to show how work is moving forward. Only the owner can mark an item complete.

Assigning work to others

  1. Edit an action item and add assignee emails in the picker (autocomplete from past assignments).
  2. New assignees receive an email notification.
  3. If they don't have an account yet, assignments link automatically when they register with that email.
  4. Remove an assignee with the × on their email badge — they are notified.

Action queue

The right sidebar shows your unified Action Queue:

  • Your own action items from saved meetings
  • Items assigned to you by others
  • Sorted: overdue first, then priority, then due date

Click an item to expand it — view progress history and add comments. Click Open meeting if you own the meeting.

Archive

Header → Archive shows completed action items with full comment history. Owners can Reopen an item to continue work.

Help & troubleshooting

Verification email not arriving

Check spam. Click Resend code — a new prefix will appear on screen; use the matching email.

Fathom not syncing

Confirm your API key in Settings. Click Refresh. If rate-limited, wait a few minutes.

Can't see meeting content

Import the meeting from the Fathom tab first. Encrypted content requires an active signed-in session.

Assigned items not showing

Register with the same email used in the assignment. Items link automatically on first login.
